
momlevel -- Sea Level Related Routines for MOM
======================================================

**momlelvel** is an open-source Python package for calculating sea
level-related fields with MOM6 output.

Quick Install
-------------
To install from Anaconda:

.. parsed-literal::
   conda install -c conda-forge momlevel

To install from PyPi:

.. parsed-literal::
   pip install momlevel
